My Projects

Smart Agricultural Field Protector (PROTOTYPE)

The Smart Agricultural Field Protector is an innovative system designed to protect farm fields from potential threats, like wild animals, using advanced sensor technologies. The system employs a sound detection sensor (KY-038) to detect environmental sounds like the ringing of bells triggered by contact with electric fences. Upon detecting sound, the system alerts farmers via LED indicators and allows the system to be remotely controlled using Bluetooth (HC-05 module). The Bluetooth feature enables farmers to arm or disarm the system, adding convenience and flexibility for remote farm management.
